🤖 Dashboard AGV | Corredor Inteligente

Este repositório contém o código do painel interativo (Dashboard) desenvolvido como Atividade Processual para a disciplina de Geometria Analítica e Álgebra Linear do 1º período de Engenharia da Faculdade MULTIVIX (Cachoeiro de Itapemirim - ES).

Esse projeto tem um pouco mais de 1 mes de pensamentos ideias e execuções (algumas equivocadas outras assertivas)
Tudo começou quando comecei a estudar os calculos para execução do projeto e fui procurar tecnologias no python para ter uma curva de aprendizado dentro da minha zona de conforto que é o estudo de tecnologias
Projeto para a Faculdade Multivix, orientado pelo Professor Leonardo

Programado em Python usando Streamlit para o deploy e aproveitando para dar uma estilizada em css (mas confeso, eu odeio front end e não manjo muito, acabei me rendendo ao Gemini para aplicação de estilização das páginas
Textos foram escritos por mim, unica coisa que o gemini fez foi a estilização e design

Cada lógica, cada calculo, cada bug, noite de sono perdida, por 3 pontos (e conhecimento)
Eu gostei de trabalhar com esse projeto por 2 fatores:
1 - adquiri conhecimento sobre o fpdf
2 - Meu primeiro projeto sozinho e que faz sentido

💻 Tecnologias Utilizadas

Python: A linguagem mãe do projeto.

Streamlit: Para criar a interface web (e me fazer sofrer misturando HTML no meio do código).

Pandas & NumPy: Para lidar com as tabelas de dados, matrizes e cálculos de Álgebra Linear.

Plotly: Para os gráficos interativos lindíssimos (adeus, Matplotlib estático!).

FPDF: Para a geração mágica do relatório final em PDF.

Gemini (IA): Arquiteto honorário do CSS.
